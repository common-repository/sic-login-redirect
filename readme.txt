== Description ==
This plugin allows you to redirect a user to any page of your choice on log-in
After enabling the plugin, you can access it (‘SIC Login Redirect’) from the settings menu. Here you can choose the redirect destination page for users who do not have author permissions.

If you select the first option in the drop down ‘Use Default Settings’, the redirect url will be the WordPress default one.

== Installation ==
1. Upload ”SIC Login Redirect” to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
